## Começando

Primeiro, Rode o comando:

```bash
npm run dev
# ou
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) com o seu navegador para ver o resultado.

Esse projeto foi feito durante um curso de React ministrado pelo professor Leornardo Leitão.
